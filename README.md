### Installation:
Make sure you have python3 and pip3 installed before running these.
Linux:
```
git clone https://github.com/SpectrTech/PSFirmManager.git
cd PSFirmManager
pip3 install -r requirements.txt
chmod +x main.py
```

### How to use:

```
./main.py
```
